<h1 align="center">Discussions</h1>

This repository is the home of GitHub Discussions for this organisation.

It provides a central place for:

- Asking questions
- Proposing ideas and improvements
- Sharing designs, concepts and early explorations
- Discussing project direction and internal standards
- Community discussions

All discussion threads, categories and related metadata are stored in this repository. No source code is maintained here.

## Local and Global Discussions

Use this repository for discussions relating to this organisation.

For ecosystem-wide discussions, cross-organisation topics, or anything relating to **The Lupaxa Project** as a whole, please use the central discussion hub:
[The Lupaxa Project Discussions](https://github.com/the-lupaxa-project/discussions)
